# Dictionary - Web page 
This is a Word Dictionary where we can search any specific word and get their pronunciation in audio, meanings of words, examples and may more things.

# Features
  * Audio pronunciation
  * Real Meanig of words
  * Examples

# TechStack
  * HTML - for structure the web design
  * CSS - for adding design and sharp colours 
  * JavaScript - for funtioning all the elements
